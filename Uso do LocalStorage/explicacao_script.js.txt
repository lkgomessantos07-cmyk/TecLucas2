Explicação do Arquivo script.js
=============================

Este arquivo JavaScript (script.js) gerencia um sistema de cadastro de funcionários, utilizando localStorage para armazenar dados. Inclui funcionalidades de adicionar, editar, remover e visualizar funcionários. Abaixo, explicação detalhada por seções.

**Visão Geral:**
- Interage com formulário HTML para entrada de dados (nome, idade, cargo, salário, departamento).
- Armazena funcionários em array no localStorage.
- Exibe funcionários em tabela com botões para ações.
- Funcionalidades: cadastrar, editar, remover, visualizar.
- Conceitos: DOM, eventos, localStorage, JSON.

**Quebra do Código por Seções:**

1. **Variável Global editIndex:**
   - `let editIndex = -1;`
   - Controla se está editando (-1 para novo, índice para editar).

2. **Função cadastrarFuncionario(e):**
   - Previne submit padrão.
   - Coleta dados do formulário em objeto 'funcionario'.
   - Se editIndex >=0, atualiza array no índice; senão, adiciona novo.
   - Salva no localStorage, recarrega tabela, reseta formulário.

3. **Função editarFuncionario(index):**
   - Carrega dados do funcionário no formulário.
   - Define editIndex e muda botão para "Atualizar".

4. **Função carregarFuncionarios():**
   - Limpa tabela.
   - Carrega array do localStorage.
   - Para cada funcionário, cria linha com dados e botões (Ver, Editar, Remover).

5. **Função mostrarFuncionario(index):**
   - Mostra dados em alert.

6. **Função removerFuncionario(index):**
   - Remove do array, salva, recarrega tabela.

7. **Evento window.onload:**
   - Carrega funcionários ao iniciar página.

**Interação com HTML:**
- Formulário com campos e botão submit.
- Tabela para exibir funcionários.
- Botões na tabela chamam funções.

**Conceitos Chave:**
- Manipulação DOM para elementos.
- Eventos para interações.
- localStorage para persistência.
- JSON para serialização.
- Arrays e objetos para dados.
